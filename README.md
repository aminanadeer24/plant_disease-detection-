# 🌿 Plant Disease Detection Using CNN
This project uses a Convolutional Neural Network (CNN) to detect plant diseases from leaf images. It helps in identifying diseases early and improving crop health.

##  Features
- Detect plant diseases from images
- Classify multiple disease types
- Fast and accurate prediction

##  Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

  ##  Dataset
- Plant leaf images dataset (e.g., PlantVillage)
- Contains healthy and diseased leaf images

  ##  How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train the model:
python train.py

3. Run prediction:
python predict.py
